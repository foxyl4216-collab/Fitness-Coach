STEP 4 — Database Schema (SQL)

Generate SQL for Supabase migration:

TABLE users
- id uuid primary key
- email text
- created_at timestamp default now()

TABLE user_profiles
- id uuid primary key
- user_id uuid references users(id)
- age int
- height int
- weight float
- goal_type text
- focus_track text
- experience_level text
- diet_preference text
- equipment_access text
- weekly_availability int
- created_at timestamp default now()

TABLE weekly_plans
- id uuid primary key
- user_id uuid
- week_number int
- calorie_target int
- workout_json jsonb
- created_at timestamp default now()

TABLE weekly_checkins
- id uuid primary key
- user_id uuid
- week_number int
- weight float
- adherence_percent int
- energy_level text
- waist_measurement float
- created_at timestamp default now()

TABLE calorie_logs
- id uuid primary key
- user_id uuid
- date date
- food_name text
- calories int
- source text
- created_at timestamp default now()

Enable Row Level Security on all tables.

Policies:
Users can only read/write their own rows.
